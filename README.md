# ProjetoCodigoparatodxs,
Programa de utilização simples, para compilar basta apenas clicar "F5", e para utilização apenas seguir as intruções em cada etapa, quando pedir para responder com "sim ou não" responder com letras minusculas, nos menus utilizar os numeros por exemplo: 1,2,3, devido a utilização do "ThreadSleep" em alguns momentos precisara aguardar alguns segundinhos.
